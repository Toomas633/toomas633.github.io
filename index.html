About this project

This is Arduino Uno robotic arm using joystick input. It's my first Arduino project that I have coded completlely by myself from the beginning.
This project is still in progress and constantly updated. I found some spare servos a year ago but never got the previous project to work. Now salvaged a broken controller for joysticks and decided to make a new robotic arm. The arm needs some spare metal or weight in the base for extra heft so it won't tip over so easily. Arm is using servos with metal gearboxes for extra sturdiness and better power and performance. Joysticks cahnge the servo position values so they hold position and won't come back when you let go.
