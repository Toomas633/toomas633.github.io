Overview

Here is a short overview of the most used Linux commands and are mostly universal to all of the different distros (aka. Ubuntu, Mint, Raspian, Debian etc.). Also here are listed some more or less advanced commands, that are relatively simple to use.

Commands

Superuser and general system commands

sudo -i                                                                 Enters root user (admin) mode.
sudo apt update                                                   Searches for package updates.
sudo apt upgrade                                                 Installs updates.
sudo apt install 'app name'                                     Installs the specified program/app.
sudo apt autoremove                                             Removes unused/outdated programs.
sudo apt clean                                                      Removes preinstall files.
sudo halt -p                                                          Shuts down the system.
sudo shutdown +5                                                Shuts the system down after 5 minutes.
sudo shutdown 9:24                                             Shuts the system down at 9:24.
sudo shutdown -r or sudo reboot                            Reboots the system.
apt -f install                                                          Force install.
ping 'address'                                                      Pings a website or an IP address (shows if there is a connection).
dd if=/dev/zero of='device'                                  Overwrites the specified disks with zeros.
time 'command'                                                   Shows how long the command will take.

Directory and file commands

cd 'directory'                                  Changes directory.
cd ..                                               Goes back a directory.
ls                                                    Lists all the files in the folder.
ls -a                                                 Lists all the files (including hidden files).
pwd                                                Shows current directory.
mkdir 'name'                                    Makes a directory with the specified name.
mkdir 'file'-{0..'num'}                         Creates the number of folders in order.
rmdir 'dir'                                        Removes directory.
cat 'file name'                                   Prints the text file insides.
nano 'file name'                                Creates/edits the file specified.
rm 'file'                                            Deletes file.
chgrp 'owner' 'dir'                             Changes the directory's owner user or own.
chmod 'numeric rule value' 'dir/file'    Changes access rules (read, write, execute). 
r=4                                                 Read numeric value.
w=2                                                Write numeric value.
x=1                                                  Execute numeric value.


Examples of access rules
Grouping of 3 root, owner, guest (rwx or any form of 3 characters is one group and so on)
770 rwxrwx---
644 rw-r--r--
650 rw-r-x---
655 rw-r-xr-x
777 rwxrwxrwx

User commands

passwd 'user'                               Changes the users password.
addgroup 'name'                          Adds a group.
adduser 'name'                            Adds user.
su 'user'                                      Changes user acc.
sudo usermod -aG 'group' 'user'    Adds user to the group (for example gives administrative rights with sudo group).
groups 'user'                                List all the groups a user is in.
usermod '..'                                 Changes user data.

Process commands

top                        Process table
htop                      Graphic process table (Task Manager alternative on Linux)(May need installation using sudo apt install htop ).
ps -aux                   Shows running processes.
wget 'url'                Download file from a url.
dpkg -i '.deb file'     Installs the downloaded and specified package.
